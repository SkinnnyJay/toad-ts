You are tasked with running the full test suite via `npm run test`.

Responsibilities:

- Run tests
- Look at the output
- Fix any failed tests. Its important you fix one test at a time and confirm each test is fixed before running a full test suite again.

Goal:

- All tests pass.

Pass:

- All tests pass and 100% success no failed test cases.
- Run `npm run build` to ensure everything compiles.

Fail:

- Tests fail to run.
- Tests are not fixed.
- Tests don't complete the full expected responsibilities.
